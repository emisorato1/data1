# Observability

Stack completo de observabilidad y monitoreo. Incluye **Langfuse** para trazabilidad de LLMs (traces, spans, tokens, latencia), **Prometheus** para recolección de métricas de sistema y aplicación, **Grafana** con dashboards predefinidos para visualización, y configuración de alertas. Gestiona el trace_id a través de todo el sistema, permitiendo debugging y análisis de performance. Todos los servicios envían telemetría a este stack para visibilidad unificada del comportamiento de agentes y APIs.
