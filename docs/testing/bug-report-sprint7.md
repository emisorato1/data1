## Criterios de Severidad

| Severidad | Descripción | SLA de Fix |

| **Crítico** | Bloquea funcionalidad core, sistema crashea, pérdida de datos | Fix en sprint actual |

| **Medio** | Experiencia degradada, workaround disponible | Backlog priorizado |
| **Bajo** | Cosmético, mejora menor, edge case raro | Backlog normal |
