# Docs

Documentación técnica y arquitectónica del proyecto. Incluye documentación **arc42** para decisiones de arquitectura, diagramas C4 de contexto y contenedores, especificaciones OpenAPI de las APIs, guías de desarrollo, runbooks operacionales, ADRs (Architecture Decision Records) y manuales de usuario. Sigue estándares profesionales para facilitar onboarding de nuevos desarrolladores, mantenimiento y compliance. Contiene también guías de integración con OpenText ECM y configuración de observabilidad.
